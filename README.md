# Data Analytics Algorithm Comparison

This project compares multiple machine learning algorithms using the Iris dataset.

Algorithms used:
- Logistic Regression
- KNN
- Decision Tree

Evaluation Metrics:
- Accuracy
- F1 Score

The project analyzes which algorithm performs best on the dataset.

Technologies used:
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook
